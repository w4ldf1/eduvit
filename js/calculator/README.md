# Calculator

Задача: реализовать примитивный калькулятор на HTML и JavaScript.

## Требования

- кнопки цифр `0...9`
- кнопки действий `/`, `*`, `+`, `-`, `=`
- поле ручного ввода
- блок истории действий
- отдельная функция для добавления записи в историю
- вычисление в отдельной функции или отдельных функциях по типам действий

Откройте `index.html` в браузере и проверьте работу кнопок.
