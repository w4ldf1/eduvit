# Conversion

Задача: переписать исходные тернарные выражения через два варианта:

- `if...else`
- `switch`

Результат должен выводиться через `console.log()`.

## Файлы

- `tern1.js` - заготовка для первого выражения.
- `tern2.js` - заготовка для второго выражения.
- `check.js` - быстрый запуск проверок.

## Проверка

```bash
node js/conversion/check.js
```

Пока функции не реализованы, проверка будет падать. После реализации оба варианта должны давать тот же результат, что и исходное тернарное выражение.
